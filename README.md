# 自动化安装脚本

一个可以让您在服务器上**更加省心**的自动化安装脚本！

## 特性

**只需一行代码，脚本将会自动安装程序：**

```sh
$ python install.py
```

## 环境

> python 2.7

> Centos6+/Debian && Centos7/Debian

脚本中部分代码使用python3执行将会出现错误,默认Centos7会自动安装python 2.7,无需手动安装。

## 用法

```sh
$ python install.py
```
